# OpenBFC - Open Source BrainFuck Compiler
A brainfuck compiler that turns brainfuck to C++ then compiles that using the g++ compiler

## How to use:
to compile a .bf all you must do is:
1. name the file you wish to compile: main.bf
2. place the main.bf in the same directory as the OpenBFC executable
3. run OpenBFC

## What you need:
1. gcc compiler properly installed
2. there is no step 2 thats it

## Demos:
In the demo folder are a few .bf's to run them simply rename and place in the same directory as the OpenBFC executable
